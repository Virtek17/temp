width - задает ширину для блочного элемента, лучше использовать max-width
height - задает высоту для блочного элемента, если уж и задаешь то лучше использовать height

color - цвет для текста, указывается в следующих форматах:
white, #ffffff, rgb(255,255,255), rgba(255,255,255, 0.9)

font-size - чтобы изменить размер шрифта
font-weight - чтобы изменить толщину шрифта

border - задать обводку для элемента, нужно указать толщину линии, тип линии, цвет
  border: 1px solid red;
  solid - прямая линия
  dashed - пунктирная линия
  dotted - точки
  double - двойная сплошная линия
Можно задать border для определенной стороны с помщью top, left, right, bottom:
border-top: 1px solid yellow; 


border-radius - если нужно скруглить углы для блока

background - свойство для работы с фоном
background-color - задать цвет фона, указывается в следующих форматах: 
white, #ffffff, rgb(255,255,255), rgba(255,255,255, 0.9)

background-img - поставить в качестве фона картинку, прописывается следующим образом
background-img: url('путь до картинки'); 

background-repeat - повторять или нет фоновую картинку, два значения (repeat, no-repeat)
background-position - чтобы располгать фоновую картинку так как тебе нужно, (left, right, top, bottom, center (можно указывать в px или %));
background-size - для изменения размеров 
background-size: auto; 
  auto - значение по умолчанию, картинка никак не изменится в размерах
  cover - изображение масштабируется без изменения пропорций так, чтобы закрыть фон всего элемента
  contain - изображение масштабируется без изменения пропорций так, чтобы полностью поместиться в элемент


padding - внутренние отступы элемента
  padding: 10px 20px 30px 40px; (верх/право/низ/лево)
  padding: 20px 15px; (верх-низ/лево-право)
  padding: 20px; (все стороны)

margin - внешние отступы элемента
  margin: 10px 20px 30px 40px; (верх/право/низ/лево)
  margin: 20px 15px; (верх-низ/лево-право)
  margin: 20px; (все стороны)


display - может изменить блочный элемент на строчный (или другой тип)
  block - элмент станет блочным, будет занимать всю ширину родительского блока, можно применять width, height, margin,padding...

  inline - элмент станет строчным, будет подстраивать свой размер под размер своего контента, нельзя задать высоту, ширину, отступы

  inline-block - элемент станет как бы строчным, но ему можно будет изменить высоту, ширину, задать отступы
  flex - чтобы выстроить элементы в строчку/столбик




position - если нужно спозиционировать элемент как то по особенному
  static - значение по умолчанию, разницу не увидишь
  relative - можно перемещать элемент относительно его первоначального положения
  absoltute - можно перемещать элемент относительно всей страницы, либо же относительно ближайшего элемента с relative
