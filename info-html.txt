div - используется для группировки элементов, помогает организовать структуру страницы (блочный) 
span - используется для выделения части текста внутри других тэгов (строчный)
p - тэг абзаца, в нем пиши какой-либо текст (блочный)
a - тэг ссылка, используется для всех ссылок на сайте в атрибут href можно прописать ссылку на другой сайт, либо через #id перейти к нужной части на этом сайте. (строчный)
h1-h6 - тэги для обозначения заголовков, должны идти по порядку, при этом h1 может быть только 1 на странице (блочные)

header - тэг который обозначает верхнюю часть страницы - шапку, обычно содежрит логоти, навигацию, меню (блочный)
main - тэг обозначающий главный контент сайта (блочный)
footer - тэг обозначающий нижнюю часть старницы - подвал, обычно содежрит контакты, ссылки (блочный)
section - тэг для логического раздела страницы, пишется внутри main (блочный) 

ul - тэг для обозначения ненумерованного списка.
ol - тэг для обозначения нумерованного списка.
li - тэг для обозначения элемента списка


button - тэг кнопки
img - тэг картинки, в атрибуте src='' указывается путь до картинки, в атрибуте alt='' пишется текст который будет отображаться в случае если картинка не загрузится


Подключение файла стилей
Чтобы подключить файл стилей в html файл нужно в тэге <head></head> прописать тэг <link/> с указанием пути до нужного css файла
Пример:
<head>
  <link rel="stylesheet" href="/styles/style.css" />
</head>
